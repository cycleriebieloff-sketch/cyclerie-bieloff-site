MISE EN LIGNE SUR VERCEL
1. Aller sur https://vercel.com
2. Créer un compte ou se connecter
3. Cliquer sur Add New > Project
4. Choisir l’option d’upload/import du dossier
5. Envoyer tout le contenu du dossier cyclerie-bieloff-site
6. Le site sera publié automatiquement

CONNEXION DU DOMAINE
1. Acheter ou gérer le domaine chez OVH / registrar
2. Dans Vercel > Project > Settings > Domains
3. Ajouter : cyclerie-bieloff.fr
4. Suivre les DNS demandés par Vercel
